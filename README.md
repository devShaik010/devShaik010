## About Me

I'm an Engineer who understands the **full lifecycle** — from designing and building, to deploying, scaling, and securing systems.

> *"Any fool can write code that a computer can understand. Good engineers write infrastructure that survives the 3 AM alert."* :)

I hold a solid foundation in both **Software Development** and **DevOps Engineering** — which means I don't just run pipelines, I understand what's running through them.

My current role is squarely in **Operations** — owning the end-to-end DevOps lifecycle within a team: infrastructure, deployments, reliability, observability, and making sure things ship on time without breaking in production. I collaborate closely across engineering to keep systems healthy, scalable, and secure.

Beyond engineering, I actively **mentor 250+ learners** through AWS Cloud Club and TSIG — conducting workshops, hands-on labs, and technical sessions. Because the best way to sharpen a skill is to teach it.
